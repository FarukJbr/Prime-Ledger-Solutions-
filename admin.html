הגדרת הרשאות אדמין
🔐 שלבים להגדרת אדמין:
  1. היכנס לפרויקט Supabase שלך
  2. לחץ על "Table Editor" → בחר טבלת "profiles"
  3. מצא את המשתמש שרוצה להיות אדמין
  4. לחץ על השורה ועדכן את השדה "role" לערך "admin"
  5. לחלופין, השתמש ב-SQL Editor:
UPDATE profiles SET role='admin' WHERE email='your@email.com';
🔌 אינטגרציות תקשורת
📬 הגדרת שרת מייל

מומלץ: Resend.com (חינמי עד 100 מייל/יום) או EmailJS. אפשרות נוספת: Supabase SMTP מובנה.

טריגרים אוטומטיים:
✓ הרשמה ✓ שחזור סיסמא ✓ אישור פגישה ✓ תזכורת תשלום
📱 הגדרת SMS

מומלץ: Twilio (אמין, גלובלי) או Vonage (Nexmo). שניהם תומכים בשליחת SMS לישראל.

💳 סליקה ישראלית

ספקי סליקה מובילים בישראל: Tranzila, Cardcom, iCredit. כולם תומכים בתשלומים חוזרים ובממשק API.

לעולם אל תאחסן נתוני כרטיסי אשראי בשרת שלך. כל עיבוד תשלומים חייב להתבצע דרך ממשק מאובטח של ספק הסליקה בלבד.
